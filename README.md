# raycast-bangumi

直接在 Raycast 中管理你的 [bangumi.tv](https://bgm.tv) 上进度。

## 功能

- **列出收藏 (List Collections):** 显示全部的收藏。
- **列出当前收藏 (List Current Collections):** 管理你的状态为「在看/在玩/在读」的条目。
- **每日放送 (Daily Calendar):** 查看每日放送。
- **搜索条目 (Search Subjects):** 在 Bangumi 上搜索。

## 安装指南

1. 在 Raycast Store 中搜索 `bangumi`，并选择安装。
2. 首次使用时，需要登录你的 Bangumi 账号。请按照提示完成登录。之后，你可以直接管理你的收藏。

## 设置

你可以在设置中自定义需要显示的分类：

- **显示书籍** (默认: `true`)
- **显示动画** (默认: `true`)
- **显示音乐** (默认: `false`)
- **显示游戏** (默认: `false`)
- **显示三次元** (默认: `true`)
